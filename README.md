# TrueCalcHub
**Accurate online calculators for finance, construction, health, salary, tax, home improvement, and everyday calculations.**

Welcome to TrueCalcHub. This repository powers https://truecalchub.com, built with Next.js 16 and TailwindCSS.

## Project Architecture
- `app/` - Next.js App Router (Pages, Layouts, API Routes)
- `components/` - React components (UI, SEO, Calculators)
- `content/` - MDX content for all calculators and blog posts
- `lib/` - Utilities and configuration
- `public/` - Static assets

## Technology Stack
- Next.js (App Router, Turbopack)
- React 19
- TailwindCSS
- Lucide React (Icons)
- MDX (Content)

## Scripts
- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run start` - Start production server
- `npm run lint` - Run ESLint
